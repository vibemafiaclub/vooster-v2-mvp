이 모든 작업을, codex goal 기능을 사용해서 할 것입니다. `/goals/{datetime}-{name}.md` 형식으로 적절히 문서를 생성해주세요. codex 에 다음과 같이 프롬프팅할 것이며, codex는 해당 목표가 완수될때까지 멈추지않고 loop할 것입니다. 우리의 의도가 정확히 반영되고, codex가 무한히 반복하며 context가 최적화되더라도 작업흐름을 잃지 않고 일관되게 작업할 수 있도록 고려하여 내용을 구성해주세요. 또, 품질게이트를 추가할 경우 이들이 너무 느리거나 과하게 작성되지 않도록 가이드도 잘 포함해주세요.
